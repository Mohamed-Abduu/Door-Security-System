# Door-Security-System
The system is a 4-bit door security system where the user enters a password using four switches and presses a CHECK button to verify it. After the button is pressed, the system compares the entered code with the stored password. If the password is correct, a green LED turns on. If the password is incorrect, a red LED and buzzer are activated.

## Technologies Used
- Tinkercad
- 74HC86 XOR Gate
- 2N2222 Transistor
- LEDs and Buzzer

## Features
- Password verification
- Green LED for access granted
- Red LED and buzzer for incorrect password
